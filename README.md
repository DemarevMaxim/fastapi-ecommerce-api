# 🛒 E-commerce API (FastAPI)

REST API для интернет-магазина, реализованный на FastAPI с использованием PostgreSQL и Docker.

## 🚀 Возможности

- Управление пользователями (Users)
- Управление товарами (Products)
- Управление категориями (Categories)
- Корзина (Cart)
- Заказы (Orders)
- CRUD операции
- PostgreSQL база данных
- Docker и Docker Compose
- Swagger документация

---

## 🛠 Технологии

- Python 3.11
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- Docker Compose
- Pydantic
